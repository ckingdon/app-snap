# app-snap

I use keyboard chords (via Mate Desktop's Keyboard Shortcuts app) to positions app windows.

For example:

    Shift+Mod4+Left
    maps to
    app-snap left

    Mod4+Right
    maps to
    app-snap right 1/2

    Mod4+Up
    maps to
    app-snap center 2/3

`display-deets` is the script that retrieves details needed by `app-snap`.
